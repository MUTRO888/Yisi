Yisi 是为 macOS 打造的智能语义转换工具。从翻译出发，让任何信息以你需要的形式呈现——无声地，不打扰地。

## 第一章：从翻译开始

大多数人需要翻译的那一刻，都是被打断的那一刻。

打开浏览器，切换标签页，粘贴文字，等待结果，再切回来——流程走完，思路已经断了。

Yisi 从这里出发。选中文字，按下快捷键，结果就在原地浮现。没有跳转，没有等待，没有上下文的切换。它的设计原则只有一条：完成这件事，然后消失。

### 翻译，做到极致

**地道的译文。** 内置经过反复打磨的翻译 Prompt，不是字对字的机械转换，而是真正可读、自然的语言输出。

**越用越懂你（Learned Rules）。** 当你纠正了一个术语，或者调整了一种表达方式，Yisi 会把这个偏好记住，变成规则，在之后的翻译里自动应用。你不需要每次重新说明，它已经知道了。

**断网也能用。** 集成 macOS 原生 System Translation，离线环境下同样可以完成翻译，数据完全在本地处理，不经过任何服务器。

---

## 第二章：然后，超越翻译

翻译是语言到语言的转换。但信息需要的转换，远不止于此。

一段代码需要被解释。一封草稿需要被润色。一张截图里的报错需要被修复。一个模糊的想法需要被扩展成 Prompt。

这些都是意思在不同形式之间的流转。Yisi 为此提供两种方式。

### Preset Mode · 预设模式

提前定义好你的工作场景，此后每次触发，无需任何多余操作。

你来定义两件事：**这段输入是什么**（输入感知），以及**你想要什么样的输出**（输出指令）。比如：

- 输入感知：一段可能存在 Bug 的 Python 代码
- 输出指令：以资深工程师的视角，找出问题并给出修复建议

配置一次，永久有效。此后选中代码，快捷键，完成——整个过程里你没有描述任何需求，没有建立任何对话，AI 只是做了它该做的事，然后退场。

**预设是你思维习惯的具象化。它不是工具的功能，而是你工作方式的延伸。**

### Custom Mode · 即时模式

当需求是一次性的，不需要提前配置。触发时，弹窗直接呈现两个填空：**这是什么**，以及**请做什么**。两个问题，分开填写，当场处理。

比如截取一个报错弹窗——"这是一个错误提示"，"请告诉我如何解决"。选中一段英文合同——"这是一份法律文件"，"请提取其中的关键条款"。

预设模式是把意图提前固化，即时模式是把同样的定义搬到了触发的那一刻。**两者的逻辑完全相同，区别只在于：你是想一劳永逸，还是随用随填。**

## 第三章：视觉也是输入

Yisi 不只处理文字。

当你的问题藏在一张图里——一份设计稿，一个数据图表，一个报错截图——Yisi 同样可以理解它。按下截图快捷键，框选屏幕上的任意区域，Preset 或 Custom 模式同样适用。

文字和图像，都是信息，都可以被转换。你不需要把图像转述成文字再处理，直接框选，直接呈现。

图片模式可以单独配置视觉模型，与文本模式各自独立，也可以选择保持一致。输入的形式由内容决定，处理的逻辑始终如一。

## 第四章：为什么是 Yisi，而不是其他 AI 工具？

Mac 上不缺 AI。快捷键唤起、截图上传、和 AI 对话——很多工具都能做到这些。

但有一件事值得想清楚：**在那些流程里，AI 始终是一个他者。**

你描述问题，它回答问题。你提供截图，它给出分析。你是提问者，它是回应者。无论体验多么流畅，这个关系的本质是：你在打断自己正在做的事，去和一个外部存在交流，然后再回来。

Yisi 从根本上打破了这种关系。

**Yisi 没有对话框。**

不是"选中内容 → 打开对话 → 描述需求 → 等待回复"，而是"选中 → 呈现"。没有来回，没有你问我答，没有需要你去维持的对话关系。你不是在和 AI 交流——你只是在做你本来要做的事，AI 悄悄完成了那个转换，然后消失了。

从界面上也能看出这种设计哲学。Yisi 的弹窗里没有对话历史，没有追问的入口，没有 AI 的名字和头像。左边是你选中的内容，右边是它的新形态。窗口关掉，你回到原来的地方，一切如常——只是那件事已经完成了。

预设模式把这个逻辑推到了极致：你甚至不需要在触发的那一刻思考任何事情。你的意图早已写进了预设，选中即触发，触发即完成。AI 是你工作流里无声的一环，不是你去拜访的对话伙伴。

这不是说对话式 AI 没有价值。深度探讨、复杂推理、多轮修改——那些场景需要对话。但当你只是想把眼前这段内容变成另一种形式，你不需要建立一段对话关系。你需要的，是一个无声完成转换的工具。

这就是 Yisi。

## 快速上手

**三步完成一次转换：**

1. **选中**：在任何应用里选中文字，或按截图快捷键框选屏幕
2. **触发**：`Cmd + C + C`（文本）／ `Cmd + Shift + X`（截图）
3. **呈现**：结果在弹窗中直接显示，关闭即返回

快捷键可在 Settings → 通用 中自定义。

**五分钟完成配置：**

Yisi 支持 Gemini、OpenAI、智谱 AI。推荐从智谱 AI 开始——新用户注册即送 20 万 tokens，足够 Yisi 用很久。

1. 前往 [智谱 AI 开放平台](https://open.bigmodel.cn) 注册并获取 API Key
2. 打开 Yisi → Settings → AI 服务
3. 提供商选 Zhipu AI，粘贴 API Key
4. 模型已预设，开箱即用。如有需要，也可以自行更换
5. 完成，开始使用
