## 📖 核心理念

Yisi 从根本上打破了传统的 AI 交互关系。**Yisi 没有对话框。**

不是「选中内容 → 打开对话 → 描述需求 → 等待回复」，而是「选中 → 呈现」。没有来回，没有你问我答，没有需要你去维持的对话关系。你不是在和 AI 交流——你只是在做你本来要做的事，AI 悄悄完成了那个转换，然后消失了。

## ✨ 主要特性

### 1. 从翻译开始，做到极致
大多数人需要翻译的那一刻，都是被打断的那一刻。Yisi 从这里出发。选中文字，按下快捷键，结果就在原地浮现。没有跳转，没有等待，没有上下文的切换。它的设计原则只有一条：完成这件事，然后消失。
- **地道的译文：** 内置经过反复打磨的翻译 Prompt，输出真正可读、自然的语言。
- **越用越懂你 (Learned Rules)：** 记住你的纠正和偏好，并在之后的翻译里自动应用。
- **断网也能用：** 集成 macOS 原生 System Translation，支持离线环境，数据完全在本地处理。

### 2. 然后，超越翻译
翻译是语言到语言的转换。但信息需要的转换，远不止于此。一段代码需要被解释。一封草稿需要被润色。一张截图里的报错需要被修复。Yisi 为此提供两种方式：
- **Preset Mode（预设模式）：** 提前定义好你的工作场景（我是什么？我想要什么？），配置一次，永久有效。此后选中代码，快捷键触发，AI 只是做了它该做的事，然后退场。
- **Custom Mode（即时模式）：** 当需求是一次性的，不需要提前配置。触发时，弹窗直接呈现两个填空：「这是什么」以及「请做什么」。当场填写，当场处理。

### 3. 视觉也是输入
当你的问题藏在一张图里（设计稿、数据图表、报错截图），Yisi 同样可以理解它。
按下截图快捷键，框选屏幕上的任意区域，预设或即时模式同样适用。文字和图像，都是信息，都可以被转换。你不需要把图像转述成文字再处理，直接框选，直接呈现。

## 🚀 快速上手

**三步完成一次转换：**

1. **选中**：在任何应用里选中文字，或按截图快捷键框选屏幕
2. **触发**：`Cmd + C + C`（文本）／ `Cmd + Shift + X`（截图）
3. **呈现**：结果在弹窗中直接显示，关闭即返回

*(快捷键可在 Settings → 通用 中自定义)*

**五分钟完成配置：**

Yisi 支持 Gemini、OpenAI、智谱 AI。推荐从智谱 AI 开始——新用户注册送极多 Tokens，足够 Yisi 用很久。

1. 前往 [智谱 AI 开放平台](https://open.bigmodel.cn) 注册并获取 API Key
2. 打开 Yisi → Settings → AI 服务
3. 提供商选 Zhipu AI，粘贴 API Key
4. 模型已预设，开箱即用。如有需要，也可以自行更换
5. 完成，完全开启无感互联体验
