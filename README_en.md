## âœ¨ Key Features

### 1. Translation, Elevated
The exact moment most people need translation is the moment they get interrupted. With Yisi, simply select text, press a shortcut, and the result emerges right there in place. No page jumping, no context switching.
- **Authentic Translations:** Built-in polished translation prompts produce highly readable, natural language, unlike mechanical word-for-word generation.
- **Learned Rules:** Whenever you correct a term or modify a tone, Yisi remembers and automatically applies it going forward.
- **Offline Support:** Built with macOS native System Translation to work securely even without an internet connection.

### 2. Beyond Translation
Code needs explaining. Drafts need polishing. Error screenshots need fixing. Yisi makes this flow smoothly via two modes:
- **Preset Mode:** Predefine your scenario (e.g., input perception + output command). Configure once, use forever. Your intent is concretized in the configuration, making the AI a smooth extension of your workflow.
- **Custom Mode:** For one-off needs. Trigger the prompt, tell Yisi exactly what the content is and what to do, and have it handled immediately. 

### 3. Vision as Input
When your question is tucked in an image (a design draft, a chart, or an error log), Yisi understands it just the same. 
Use the screenshot shortcut to crop any area on your screen, and the Preset or Custom mode will analyze the visual data silently.
